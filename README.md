## How to Run Locally

1. Clone the repo on you local(`https://github.com/Satyajit-Sahoo45/stratforge-assignment.git`)

3. Install dependencies:

### `npm install`

3. Update .env
### `add DATABASE_URL, JWT_SECRET, PORT`

4. Install dependencies:

### `npm run build`

5. Start the development server:

### `npm run server`
