## How to Run Locally

1. Clone the repo on your local(`https://github.com/Satyajit-Sahoo45/td-backend.git`)

3. Install dependencies:

   ### `npm install`

3. Update .env
   ### `add DATABASE_URL, JWT_SECRET and PORT`

4. Install dependencies:

   ### `npm run build`

5. Start the development server:

   ### `npm run server`
